# Chennai Flood Simulator 🌊
Real-Time Hydrological Decision Intelligence Tool
By Dr. Sundararaman Chintamani | sundarspeaks.com

## About
A free browser-based tool simulating 15 Chennai flood scenarios for urban flood risk awareness, emergency planning and policy intelligence. Validated against NDMA Tamil Nadu Floods report, IMD rainfall records, Climate Central 2050 projections and OpenStreetMap data.

## Features
- 15 flood scenarios (NE/SW Monsoon, Cyclone, Storm Surge, Reservoir Release, Climate 2050, Cloudburst, Tidal Lock, Governance Failure)
- Live Esri satellite map with OpenStreetMap waterways
- Decision Tool: 4-trigger combination → flood sequence + impact map
- Audio narration in Indian English
- 6-belt vulnerability framework + scenario × area risk matrix
- Depth in feet and inches
